JavaScript linking

To connect an HTML file to a JavaScript file, you will need to use a script tag in your HTML file.
Inside the script tag, you can either write your JavaScript code directly or you can refer to an external JavaScript file using the src attribute.

JavaScript variables

JavaScript variables can be declared in 4 ways:

  • Automatically
  • Using var
  • Using let
  • Using const

Variables are containers for storing data.

Variable Declaration using the keyword let:

let numberA = 10;

console.log('Tall A:'); (Logging a string)
console.log(numberA); (Logging a variable)

let numberB = 32;

console.log('Tall B:'); (Logging a string)
console.log(numberB); (Logging a variable)

let numberC = numberA / numberB + numberA - numberB;

console.log('Tall C:');
console.log(numberC);

Different types of variables in JavaScript:

  • String - stores text, such as "Hello". String values are surrounded by double quotes
  • int - stores integers (whole numbers), without decimals, such as 123 or -123
  • float - stores floating point numbers, with decimals, such as 19.99 or -19.99
  • char - stores single characters, such as 'a' or 'B'. Char values are surrounded by single quotes
  • boolean - stores values with two states: true or false

Javascript Console log method

The log() method writes (logs) a message to the console.

The log() method is useful for testing purposes.

Exsample 1:

console.log("Hello world!");

This line uses the console.log() function to print the string "Hello world!" to the console.

Exsample 2:

let greeting = "Hello world!";
console.log(greeting);

This line logs the value of the variable greeting to the console.

JavaScript functions and events

A JavaScript function is a block of JavaScript code, that can be executed when "called" for.
For example, a function can be called when an event occurs, like when the user clicks a button.
